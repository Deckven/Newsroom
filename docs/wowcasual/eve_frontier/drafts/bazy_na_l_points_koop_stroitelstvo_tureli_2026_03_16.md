# Базы на L-Points: кооп-строительство, турели

Строительство баз — одна из ключевых механик EVE Frontier. В Cycle 5 система получила масштабное обновление: кооперативное строительство на L-Points, Construction Sites для совместной доставки материалов и три типа специализированных турелей для обороны. Разберём процесс от выбора точки до полноценной укреплённой базы.

## L-Points: что это и почему они важны

L-Points — это специальные точки в пространстве, пригодные для размещения инфраструктуры. Не каждая координата в космосе подходит для строительства — L-Points определяют, где именно можно закрепить структуры.

### Ключевое нововведение Cycle 5

Несколько игроков теперь могут размещать свои структуры на одном L-Point. Это фундаментальное изменение, которое одновременно стимулирует кооперацию и создаёт потенциал для конфликта.

На практике это означает:
- Tribe может построить полноценную базу, где каждый участник вносит свои структуры
- Соседство на L-Point требует либо доверия, либо обороноспособности
- Стратегически важные L-Points становятся объектами борьбы

## Шаг 1: Выбор L-Point

### Критерии выбора

1. **Безопасность** — насколько активна система, есть ли враждебные Tribes поблизости
2. **Ресурсы** — близость к Orbital Zones и рифтам для снабжения
3. **Логистика** — доступность для союзников, расстояние до торговых маршрутов
4. **Стратегическое положение** — контроль над ключевыми переходами
5. **Соседство** — кто уже построил на этом или соседних L-Points

### Рекомендации

- Для первой базы выбирайте L-Point в относительно безопасной системе
- Убедитесь, что поблизости есть Orbital Zones для ресурсов
- Проверьте, нет ли враждебных структур на том же L-Point
- Обсудите выбор с Tribe — база строится совместно

## Шаг 2: Network Node и Construction Site

### Network Node

Network Node — это основа любой базы. Это первая структура, которую нужно разместить. Она определяет принадлежность базы и открывает доступ к дальнейшему строительству.

### Construction Site

Construction Site — новая механика кооперативного строительства. Это точка на принадлежащем вам Network Node, куда несколько игроков могут доставлять материалы для возведения структур.

Как работает:
1. Владелец Network Node создаёт Construction Site для конкретной структуры
2. Система показывает список необходимых материалов
3. Любой авторизованный игрок может доставить часть материалов
4. Когда все материалы собраны — структура появляется

### Преимущества кооперативного строительства

- Распределение нагрузки: никто не должен добывать все ресурсы в одиночку
- Скорость: пять игроков построят базу в пять раз быстрее одного
- Специализация: каждый добывает то, что умеет лучше
- Сплочение Tribe: совместная работа укрепляет связи

## Шаг 3: Размещение Smart Assemblies

После создания Network Node размещайте ключевые структуры:

### Приоритет 1: Производство и хранение

- **Smart Storage Unit (SSU)** — хранилище для ресурсов и предметов
- **Nursery** — производство Shell (критически важно)
- **Nest** — хранилище для запасных Shell

### Приоритет 2: Инфраструктура

- **Smart Gate** — врата для быстрого перемещения союзников
- **Дополнительные SSU** — торговые терминалы, квестовые системы

### Приоритет 3: Оборона

- **Smart Turrets** — автоматизированная защита (подробнее ниже)
- **Стены** — барьеры для замедления атакующих

## Шаг 4: Оборонительная система — три типа турелей

### Установка турелей

Smart Turret размещается в радиусе 25 км от другой Smart Assembly и автоматически прикрепляется к ней. Каждая турель оснащается одним из трёх типов вооружения.

### Autocannon Turret

| Параметр | Значение |
|----------|---------|
| Назначение | Противодействие лёгким кораблям |
| Скорострельность | Высокая |
| Урон за выстрел | Низкий |
| Эффективность | Фрегаты, малые корабли |

Autocannon — первая линия обороны. Высокая скорострельность компенсирует низкий урон и позволяет отслеживать быстрые цели. Эффективна против фрегатов вроде Exclave, которые пытаются прорваться к базе.

### Plasma Turret

| Параметр | Значение |
|----------|---------|
| Назначение | Противодействие среднему классу |
| Скорострельность | Средняя |
| Урон за выстрел | Средний |
| Эффективность | Крейсеры, средние корабли |

Plasma — универсальная турель. Баланс между скоростью стрельбы и уроном делает её надёжным выбором против наиболее распространённых типов кораблей.

### Railgun Turret

| Параметр | Значение |
|----------|---------|
| Назначение | Противодействие тяжёлым кораблям |
| Скорострельность | Низкая |
| Урон за выстрел | Высокий |
| Эффективность | Линкоры, крупные суда |

Railgun — тяжёлое оружие для отражения серьёзных атак. Медленная перезарядка компенсируется разрушительным уроном по крупным целям. Бесполезна против фрегатов — они слишком быстры.

### Рекомендуемая конфигурация обороны

Для сбалансированной защиты базы:
- 2-3 Autocannon Turret — по периметру, против быстрых рейдеров
- 1-2 Plasma Turret — в средней зоне, универсальное покрытие
- 1 Railgun Turret — в центре, против тяжёлых атакующих

## Шаг 5: Программирование логики турелей

Smart Turrets — это не просто «стрелялки». Их поведение определяется смарт-контрактом, который можно настроить.

### Базовые режимы

1. **Агрессивный** — стреляет по всем, кроме белого списка
2. **Оборонительный** — стреляет только по тем, кто атаковал первым
3. **Территориальный** — предупреждение, затем атака при дальнейшем приближении

### Продвинутая логика

- Приоритизация целей по классу корабля
- Координация между несколькими турелями (фокус-огонь)
- Экономическая логика: предложение «откупиться» перед атакой
- Интеграция с Smart Gate: блокировка враждебных кораблей на входе

### Блюпринты

Если вы не программист — используйте готовые блюпринты из сообщества. Множество конфигураций турелей доступно бесплатно. Установка блюпринта — вопрос нескольких кликов.

## Шаг 6: Стены и периметр

Стены — это барьеры, замедляющие атакующих и определяющие периметр базы. Они не неразрушимы, но дают время турелям для поражения целей.

Рекомендации:
- Размещайте стены так, чтобы создать «канал» — зону, через которую атакующим придётся лететь под огнём турелей
- Оставляйте один вход, контролируемый Smart Gate
- Не забывайте о резервном выходе для эвакуации

## Чек-лист строительства базы

1. [ ] Выбрать L-Point с учётом безопасности, ресурсов и логистики
2. [ ] Разместить Network Node
3. [ ] Создать Construction Sites для ключевых структур
4. [ ] Скоординировать доставку материалов с Tribe
5. [ ] Установить SSU для хранения ресурсов
6. [ ] Установить Nursery и Nest
7. [ ] Разместить Smart Gate для быстрого доступа
8. [ ] Установить турели: Autocannon по периметру, Plasma в средней зоне, Railgun в центре
9. [ ] Настроить логику турелей (или загрузить блюпринт)
10. [ ] Установить стены и определить периметр
11. [ ] Провести тестовую атаку для проверки обороны

Строительство базы — не разовое действие, а непрерывный процесс. Расширяйте, улучшайте, адаптируйте. Frontier вознаграждает подготовленных.
