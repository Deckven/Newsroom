# Shroud of Fear: первые шаги в Cycle 5 — Shell Industry, Orbital Zones

EVE Frontier вступила в пятый цикл раннего доступа — Shroud of Fear. Обновление вышло 11 марта 2026 года и принесло фундаментальные изменения: новую систему клонов Shell Industry, переработанные PvE-зоны, миграцию на блокчейн Sui и ребаланс кораблей. Этот гайд поможет новичкам разобраться в основах и не погибнуть в первый же час.

## Что такое Cycles

EVE Frontier находится в стадии Founder Access — раннего доступа для основателей. Разработка идёт циклами: каждый Cycle — это ограниченный по времени этап тестирования с новыми механиками. В конце каждого цикла вселенная сбрасывается, но заработанные EVE Points (очки наследия) сохраняются навсегда.

Cycle 5: Shroud of Fear — самое крупное обновление на данный момент. Впервые введена система клонов, переработан PvE, и игра перешла с Ethereum на блокчейн Sui.

## Доступ к игре

В Cycle 5 появились два варианта доступа:

- **Founder Access** — полный доступ, включая блокчейн-функции и Smart Assemblies
- **Game-Only Pass** ($20) — доступ к геймплею без необходимости криптокошелька

Game-Only Pass — это шаг навстречу игрокам, не заинтересованным в блокчейне. Вы можете летать, стрелять, добывать ресурсы и строить без взаимодействия с криптовалютной инфраструктурой.

## Первые шаги: создание Shell

В Cycle 5 ваш персонаж — это Shell, клон-оболочка. Вот что нужно знать:

### 1. Стартовая точка

После входа в игру вы появляетесь в стартовой системе в базовом Shell. У вас есть начальный корабль и минимальный набор ресурсов.

### 2. Управление кораблём

С Cycle 4 в игре действует WASD-управление: W/S — ускорение/торможение, A/D — повороты, E/Q — управление тягой, Ctrl+Space — полная остановка. Управление стало прямым и интуитивным, ближе к космическим симуляторам, чем к классическому EVE Online.

Новые камеры в Cycle 5:
- **Chase camera (C)** — мягко следует за кораблём сзади, кинематографичная
- **Tracking camera (T)** — удерживает в кадре и ваш корабль, и выбранную цель
- Обновлённый **телескоп** — значительно более плавный зум
- В режиме телескопа виден прицел для будущего **manual targeting mode** (пока не включён)

Стартовый корабль теперь поставляется с модулем **Skip** для быстрого перемещения в орбитальных зонах. Два других модуля — **Hop** (ближе) и **Leap** (дальше) — нужно производить.

### 3. Shell Industry: основы

Shell Industry — ключевая новая система. Ваш клон — это изготовленный объект с ограниченным сроком полезности. Вот ключевые элементы:

- **Nursery** (Питомник) — производственный модуль для создания новых Shell (~10 часов на один)
- **Nest** (Гнездо) — хранилище для запасных Shell
- **Crowns** (Короны) — создаются из memories через Ascension Protocol, устанавливаются в Shell

Три новых производимых шелла: **Aggressive** (боевой), **Reaping** (добыча), **Rugged** (выносливость). Базовый incarnation shell по-прежнему бесплатен.

Критически важно: если ваш Shell уничтожен, все Crowns (навыки), установленные в него, теряются навсегда. Это создаёт реальные последствия гибели и заставляет думать о риске перед каждым вылетом.

### 4. Рекомендация для новичков

Не устанавливайте ценные Crowns в Shell, с которым отправляетесь в опасные зоны. Держите запасные Shell в Nest. Производите новые Shell заблаговременно — Nursery требует время и ресурсы.

## Orbital Zones: новые PvE-зоны

Orbital Zones заменили старую систему данжей. Вместо инстансированных подземелий — открытые, персистентные экосистемы, разбросанные по солнечным системам.

### Что изменилось

- Ресурсы, NPC и лут распределены натуралистично по зонам вокруг планет
- Feral AI (дикий ИИ) динамически патрулирует зоны, реагируя на активность
- Зоны не инстанцированы — другие игроки могут быть рядом
- Лут и ресурсы уникальны для каждой зоны

### Советы по Orbital Zones

1. **Passive Observation System** — новая система Cycle 5. Ваш корабль автоматически обнаруживает объекты по массе (гравитационные сигнатуры). Лёгкий корабль увидит тяжёлый раньше — используйте это для разведки
2. **Следите за NPC** — они полностью переработаны. NPC теперь имеют цели, память и перемещаются между точками интереса, включая базы игроков. Лут — из обломков и кэшей, не с NPC напрямую
3. **Будьте готовы к PvP** — Orbital Zones не защищены. Другие игроки могут атаковать вас, пока вы заняты NPC
4. **Начинайте с безопасных систем** — зоны различаются по уровню опасности

## Ресурсы и топливо

### Crude Matter и микро-рифты

**Микро-рифты** — новый тип рифтов, есть практически в каждой системе. Дают небольшое количество crude невысокого качества — достаточно для базового выживания и перемещения, но не для промышленности. Как сказал CCP Yoten: «Это как собирать ягоды в лесу — продержишься, но до промышленной фермы далеко».

Два полноценных типа рифтов также доступны для серьёзной добычи.

### Топливо

Топливо в Cycle 5 получило новые свойства, влияющие на потребление и межзвёздные перелёты. Следите за расходом — остаться без топлива в опасной системе равносильно гибели.

## Строительство базы

В Cycle 5 можно строить базы на L-Points — специальных точках в пространстве. Несколько игроков могут размещать структуры на одном L-Point, что стимулирует кооперацию.

### Для новичка

Не спешите строить. Сначала:
1. Освойте управление и навигацию
2. Накопите ресурсы в Orbital Zones
3. Найдите Tribe (клан) — строительство в одиночку крайне сложно
4. Изучите Construction Sites — точки кооперативной доставки материалов

## Блокчейн Sui: что нужно знать

Cycle 5 — первый цикл на блокчейне Sui (ранее игра работала на Ethereum). Для обычного геймплея это прозрачно: вы не заметите блокчейн в повседневных действиях.

Блокчейн становится важен, если вы хотите:
- Размещать Smart Assemblies (программируемые структуры)
- Торговать предметами за пределами игры
- Участвовать в хакатоне ($80,000 призовой фонд, до 31 марта)

Если блокчейн вас не интересует — Game-Only Pass за $20 даёт полноценный игровой опыт без криптокошелька.

## Чек-лист первого дня

- [ ] Освоить WASD-управление в безопасной зоне
- [ ] Попробовать новые камеры: Chase (C) и Tracking (T)
- [ ] Разобраться с интерфейсом Shell Industry и Ascension Protocol
- [ ] Создать запасной Shell в Nursery (учтите ~10ч производства)
- [ ] Изучить Passive Observation System — следите за гравитационными сигнатурами
- [ ] Использовать модуль Skip для перемещения в орбитальных зонах
- [ ] Посетить ближайшую Orbital Zone, собрать ресурсы из обломков и кэшей
- [ ] Найти Tribe для совместной игры
- [ ] Не устанавливать ценные Crowns в боевой Shell

Frontier — жёсткая игра. Здесь нет страховки, нет руки разработчика, протянутой для спасения. Но именно это делает каждое достижение настоящим.
