# Shell Industry: Nursery, Nest и Crowns — как не потерять скиллы навсегда

Shell Industry — самое значительное нововведение Cycle 5. Система позволяет игрокам производить и хранить клон-оболочки (Shell), управлять навыками через Crowns и впервые создаёт реальные последствия гибели. В этом гайде — пошаговое руководство по всем элементам системы.

## Концепция: ваше тело — расходный материал

В EVE Frontier вы — не бессмертный герой. Вы — сознание, помещённое в клон-оболочку (Shell). Shell можно произвести, улучшить, потерять. Если ваш Shell уничтожен — вы возрождаетесь в новом, но всё, что было установлено в старом, утрачивается безвозвратно.

Это фундаментально отличается от классического EVE Online, где клон-смерть была неудобством, но не катастрофой. Здесь потеря Shell — потеря навыков. Нет страховки, нет отката. Frontier не прощает ошибок.

## Шаг 1: Nursery — производство Shell

### Что это

Nursery (Питомник) — производственный модуль, отвечающий за создание новых Shell. Это ваша фабрика клонов.

### Как работает

1. Разместите Nursery на вашей базе или в привязке к Smart Assembly
2. Загрузите необходимые ресурсы — каждый Shell требует определённый набор материалов
3. Запустите производство — процесс занимает реальное время
4. После завершения Shell появляется в инвентаре, готовый к размещению в Nest

### Практические рекомендации

- Всегда держите Nursery в работе. Производство Shell должно быть непрерывным процессом
- Заранее запасайте ресурсы для производства — перебои в снабжении могут оставить вас без запасных тел
- Базовый Shell дёшев в производстве. Начинайте с него, прежде чем переходить к улучшенным вариантам
- Учитывайте время производства при планировании рискованных операций

## Шаг 2: Nest — хранилище Shell

### Что это

Nest (Гнездо) — хранилище для готовых Shell. Это ваш личный арсенал запасных тел.

### Как работает

1. Произведённые в Nursery Shell перемещаются в Nest
2. Nest хранит несколько Shell одновременно
3. При гибели текущего Shell вы автоматически активируете следующий из Nest
4. Shell в Nest можно предварительно настроить — установить Crowns до активации

### Практические рекомендации

- Никогда не оставайтесь с одним Shell. Минимум два-три запасных в Nest — правило выживания
- Размещайте Nest в безопасном месте — его потеря означает потерю всех хранящихся Shell
- Настраивайте разные Shell под разные задачи: боевой, исследовательский, транспортный
- При присоединении к Tribe обсудите размещение общего Nest на базе клана

## Шаг 3: Crowns — система навыков

### Что это

Crowns (Короны) — это модули памяти, хранящие навыки и специализации. Они устанавливаются в Shell и определяют, что именно ваш персонаж умеет делать.

### Как работает система

1. Crowns — это отдельные объекты, которые можно получить, произвести или обменять
2. Crown устанавливается в слот Shell — после установки навык активен
3. Каждый Shell имеет ограниченное количество слотов для Crowns
4. При уничтожении Shell все установленные Crowns теряются навсегда

### Типы Crowns (на основе доступной информации)

Crowns различаются по специализации. Они оптимизируют различные аспекты геймплея: пилотирование, бой, добычу ресурсов, производство, торговлю. Конкретный набор Crowns определяет вашу роль в экосистеме.

### Критически важное правило

**Crowns уничтожаются вместе с Shell.** Это ключевая механика, которая создаёт реальные ставки в каждом столкновении. Потеряв Shell с редкими Crowns, вы теряете не только время, но и конкретные возможности.

## Стратегии управления рисками

### Стратегия 1: Разделение ценности

Не ставьте все лучшие Crowns в один Shell. Распределяйте ценность между несколькими оболочками. Если потеряете одну — не потеряете всё.

### Стратегия 2: Боевой / мирный Shell

Держите два типа Shell:
- **Боевой** — базовые Crowns, которые не жалко потерять. Для PvP, Orbital Zones, рискованных вылетов
- **Мирный** — ценные Crowns для производства, торговли, программирования. Используйте только в безопасных зонах

### Стратегия 3: Резервный фонд

Храните запас Crowns отдельно от Shell. Если потеряли Shell — устанавливайте Crowns из резерва в новое тело. Это требует дисциплины, но спасает от катастрофических потерь.

### Стратегия 4: Племенная страховка

В рамках Tribe можно организовать систему взаимопомощи: общий запас Shell и Crowns, из которого члены клана могут восстановиться после потерь. Это стимулирует кооперацию и лояльность.

## Экономика Shell Industry

Shell Industry создаёт новый экономический слой:

- **Производители Shell** — игроки, специализирующиеся на массовом производстве клонов
- **Крафтеры Crowns** — создатели модулей навыков, потенциально самый прибыльный бизнес
- **Операторы Nest** — предоставление услуг хранения для других игроков
- **Страховщики** — неформальные соглашения о компенсации потерь

Спрос на Shell и Crowns постоянен: игроки гибнут, навыки теряются, нужны новые. Это создаёт устойчивый экономический цикл, привязанный к основному геймплею.

## Пошаговый чек-лист для нового игрока

1. [ ] Найти и активировать Nursery
2. [ ] Произвести первый запасной Shell
3. [ ] Разместить Nest в безопасной локации
4. [ ] Поместить запасной Shell в Nest
5. [ ] Получить базовые Crowns
6. [ ] Установить дешёвые Crowns в активный Shell для первых вылетов
7. [ ] Хранить ценные Crowns отдельно до понимания рисков
8. [ ] Начать постоянное производство Shell — поставить на поток
9. [ ] Присоединиться к Tribe для доступа к общим ресурсам

## Главное правило

Shell Industry строится на одном принципе: подготовка. В EVE Frontier побеждает не тот, кто лучше стреляет, а тот, кто лучше подготовился к потерям. Nursery работает — Nest полон — Crowns распределены — вы готовы к Frontier.
