# Smart Assemblies: программируй вселенную — SSU, Gates, Turrets

Smart Assemblies — это сердце EVE Frontier. Программируемые структуры, размещённые в космосе, которые работают на смарт-контрактах и позволяют игрокам буквально создавать правила вселенной. В этом гайде разберём все типы Smart Assemblies, их возможности и примеры применения.

## Что такое Smart Assembly

Smart Assembly — это физический программируемый объект, закреплённый в пространстве. Каждая такая структура существует на блокчейне Sui как уникальный объект, управляемый смарт-контрактом на языке Move.

Важно понимать: Smart Assemblies — это не просто здания. Это фундамент инфраструктурного слоя, который игроки проектируют, модифицируют, автоматизируют и оптимизируют. Они подчиняются неизменным законам игровой физики, но их внутренняя логика полностью в руках владельца.

Для размещения Smart Assembly нужны ресурсы и подходящая локация. После деплоя структура становится активной и начинает выполнять заложенную в неё программу.

## Smart Storage Unit (SSU)

### Базовое описание

Smart Storage Unit — самая универсальная структура в игре. По умолчанию это хранилище, размещённое в фиксированной точке космоса. Внутри — активный инвентарь, который может принимать и выдавать любые предметы.

### Возможности программирования

SSU — это чистый холст. В зависимости от смарт-контракта, загруженного владельцем, одна и та же структура может стать:

- **Торговым терминалом** — принимает ресурсы и выдаёт товары по заданному курсу
- **Квестовым NPC** — выдаёт задания и награждает за выполнение условий
- **Системой баунти** — назначает награды за уничтожение определённых целей
- **Аркадным автоматом** — мини-игры на блокчейне
- **Банком** — принимает депозиты и выдаёт проценты
- **Информационной доской** — транслирует данные другим игрокам

### Практический пример

Допустим, вы хотите создать торговый пост на оживлённом маршруте. Размещаете SSU, загружаете смарт-контракт маркетплейса, заполняете инвентарь товарами. Теперь любой пролетающий мимо игрок может обменять ресурсы на ваши товары — автоматически, без вашего присутствия. Контракт гарантирует честность сделки.

## Smart Gate

### Базовое описание

Smart Gate — это игрок-созданные межзвёздные врата. Они позволяют кораблям прыгать из одной системы в другую, минуя стандартные маршруты. Это одновременно инструмент логистики и средство контроля территории.

### Возможности программирования

Ключевая особенность Smart Gate — настраиваемая логика допуска. Владелец определяет, кто может пользоваться вратами:

- **Открытые врата** — доступны всем. Хороший вариант для торговых хабов
- **Племенные врата** — пропускают только членов вашего Tribe
- **Синдикатные врата** — доступ для членов Syndicate (объединения племён)
- **Платные врата** — проход за плату в ресурсах или топливе
- **Условные врата** — доступ по выполнению определённого условия (наличие предмета, уровень репутации)

### Стратегическое значение

Сеть Smart Gates создаёт альтернативную карту вселенной. Племя, контролирующее цепочку врат, контролирует логистику целого региона. Знание тайных маршрутов становится стратегическим ресурсом. Это прямое наследие jump-бриджей из EVE Online, но с принципиально новым уровнем гибкости.

## Smart Turret

### Базовое описание

Smart Turret — автоматизированная оборонительная структура. Устанавливается в радиусе 25 км от другой Smart Assembly и автоматически прикрепляется к ней. Оснащается одним из трёх типов вооружения.

### Типы турелей

| Тип | Назначение | Оптимальная цель |
|-----|-----------|-----------------|
| Autocannon Turret | Скорострельное оружие | Лёгкие корабли, фрегаты |
| Plasma Turret | Среднее вооружение | Корабли среднего класса |
| Railgun Turret | Тяжёлое вооружение | Крупные боевые корабли |

### Возможности программирования

Самое интересное — логика наведения. По умолчанию турель стреляет по всем вражеским целям. Но смарт-контракт позволяет задать произвольные правила:

- **Приоритет целей** — сначала лёгкие корабли, потом тяжёлые (или наоборот)
- **Белый список** — не стрелять по союзникам и нейтралам
- **Зона контроля** — агрессивное поведение только в определённом радиусе
- **Условная агрессия** — атака только тех, кто уже начал враждебные действия
- **Экономическая логика** — предупреждение и предложение «откупиться» перед атакой

### Практический пример

Вы строите базу на L-Point и хотите защитить её, но не отпугивать торговцев. Программируете турель: белый список для членов Tribe, нейтральное поведение для всех остальных (предупреждение при приближении), агрессия только против игроков с активным оружием. Автоматическая оборона без ложных срабатываний.

## Блюпринты: код для тех, кто не кодит

Не обязательно быть программистом, чтобы использовать Smart Assemblies. Экосистема EVE Frontier включает систему блюпринтов — готовых, оптимизированных пакетов на Move, созданных опытными разработчиками.

Блюпринт — это предготовый смарт-контракт, который можно развернуть на своей структуре без единой строки кода. Сообщество активно создаёт и делится блюпринтами: маркетплейсы, системы защиты, квестовые механики, логистические решения.

Репозиторий builder-examples на GitHub содержит образцы для обучения. CCP также проводит хакатоны: текущий EVE Frontier x Sui Hackathon 2026 (11-31 марта) с призовым фондом $80,000 стимулирует создание новых блюпринтов.

## Внешние приложения через API

Smart Assemblies — не единственный способ расширить вселенную. EVE Frontier предоставляет API для внешних приложений, которые могут читать и реагировать на данные игрового мира в реальном времени:

- **Карты** — интерактивные карты вселенной с актуальным состоянием
- **Дашборды** — мониторинг экономики, боевых действий, ресурсов
- **Координационные платформы** — инструменты для управления Tribes и Syndicates
- **Аналитические сервисы** — статистика, графики, прогнозы

## С чего начать

1. Изучите документацию на docs.evefrontier.com
2. Познакомьтесь с языком Move через официальные туториалы Sui
3. Просмотрите примеры в репозитории builder-examples
4. Начните с простого SSU — торгового терминала или информационной доски
5. Экспериментируйте с логикой турелей — это самый наглядный способ увидеть результат программирования

Smart Assemblies — это не побочная функция EVE Frontier. Это её суть. Вселенная, которую строят игроки, по правилам, которые пишут игроки.
