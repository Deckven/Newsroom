# Verso: End Bringer и Perfection — соло-керри для челленджей

Verso — протагонист Clair Obscur: Expedition 33 и, при правильной сборке, самый стабильный дамагер для соло-контента и челленджей. Его система Perfection Rank превращает каждое успешное парирование в рост урона, а End Bringer — в финишер, способный уничтожить любого босса. Разберём его арсенал.

## Система Perfection Rank

### Как работает

По мере боя Verso накапливает Perfection через три источника:
- **Атаки** — каждая атака и умение добавляют Perfection
- **Парирование и уклонение** — успешные защитные действия дают Perfection
- **Умения** — некоторые умения дают бонусные очки Perfection

### Ранги

Perfection делится на ранги: C → B → A → S. Каждый ранг увеличивает урон всех действий Verso. На ранге S урон достигает максимума, и многие умения получают дополнительные бонусы.

### Потеря ранга

Получение урона снижает Perfection. Это создаёт прямую связь между навыком парирования и уроном Verso: если вы парируете всё — Perfection растёт. Если пропускаете удары — падает.

### Ключевой принцип

Verso награждает мастерство боевой системы. Чем лучше вы парируете, тем больше урона наносите. Это делает его идеальным персонажем для соло-челленджей, где всё зависит от навыка игрока.

## Каталог ключевых умений

### End Bringer

**Тип**: одиночная цель, финишер
**Механика**: 6 ударов по одной цели. Если враг оглушён (stunned), урон каждого удара значительно увеличен. На ранге A оглушение цели продлевается.

**Когда использовать**: после Break или Stun. End Bringer — умение, которое вы используете для добивания босса после подготовки командой.

### Overload

**Тип**: самобафф
**Механика**: HP Verso падает до 1, но он мгновенно получает ранг A и полностью восстанавливает AP.

**Риск/Награда**: один удар убьёт Verso, но взамен вы получаете полный ход с максимальным рангом. Идеально для открывающего хода: Overload → серия атак с рангом A.

### Phantom Stars

**Тип**: AoE-атака
**Механика**: урон по всем врагам с бонусом на высоких рангах Perfection.

**Когда использовать**: против групп врагов для быстрой зачистки или для добивания оглушённых целей.

### Follow-Up

**Тип**: одиночная цель, нюк
**Механика**: экстремальный урон по одной цели. Предназначен для добивания после End Bringer, если враг выжил.

**Когда использовать**: если End Bringer не хватило — Follow-Up завершает дело.

## Каталог билдов

### Билд 1: Overload Opener (Стандартный DPS)

**Концепция**: взрывной старт через Overload с последующим градом атак на ранге A/S.

**Оружие**: Chevalam
**Скейлинг**: Agility и Luck
**Атрибуты**: Agility (приоритет) > Luck > Defence

**Ротация**:
1. Overload → HP=1, Rank A, AP полные
2. Free Aim (несколько выстрелов) → набор ранга S + стаки Burn/Mark
3. Phantom Stars (если группа) или End Bringer (если босс)
4. Follow-Up (если цель выжила)

**Риски**: после Overload Verso на 1 HP. Любой пропущенный удар = смерть. Парирование обязательно.

### Билд 2: Solo Carry (Endless Tower / Челленджи)

**Концепция**: максимальная автономность. Verso справляется сам, не завися от саппорта.

**Оружие**: Chevalam
**Атрибуты**: Agility > Luck > Vitality

**Ключевые Pictos**:
- Восстановление HP при парировании
- Бонус урона при ранге S
- Бонус AP при Perfect Parry

**Ротация**:
1. Free Aim → набор Perfection через стрельбу
2. Парирование вражеских атак → рост ранга + восстановление HP через Pictos
3. End Bringer при ранге S → максимальный урон
4. Поддержание ранга через стабильное парирование

**Преимущество**: не нужен Overload (и связанный с ним риск). Verso набирает ранг естественно через идеальную игру.

### Билд 3: One-Shot Boss Killer (Команда)

**Концепция**: команда готовит цель, Verso уничтожает за один ход.

**Оружие**: Chevalam
**Атрибуты**: Luck (максимум) > Agility > Defence

**Setup команды**:
1. Густав: Marking Shot → +50% урона
2. Sciel: Foretell стаки (опционально)
3. Любой персонаж: Break через Free Aim

**Ход Verso**:
1. Overload → Rank A, полные AP
2. Free Aim → Rank S
3. End Bringer по маркированной/оглушённой цели = миллионы урона

### Билд 4: DLC Verso (Verso's Drafts)

**Концепция**: использование нового контента из DLC для усиления Verso.

**Новые оружия**: DLC добавляет новые варианты оружия, расширяющие возможности билдов.

**Новые Pictos**: DLC-пикосы дают дополнительные бонусы для Perfection-системы и End Bringer.

**Ротация**: аналогична стандартным билдам, но с усиленными числами благодаря DLC-снаряжению.

## Chevalam: почему это лучшее оружие

**Скейлинг**: Agility и Luck

**Пассив**: позволяет начинать бой с ранга S при достаточно высоких статах.

**Синергия**: Agility = больше ходов = больше Perfection = больше времени на ранге S. Luck = больше критов = больше урона на высоких рангах.

Chevalam делает Verso самодостаточным: высокий ранг с самого начала, много ходов, высокий крит — всё, что нужно для соло-игры.

## Связь парирования и DPS

В отличие от других персонажей, у Verso защита напрямую = атака:
- Успешное парирование → +Perfection → +ранг → +урон
- Пропущенный удар → -Perfection → -ранг → -урон

Это означает, что улучшение навыка парирования буквально увеличивает DPS Verso. Нет другого персонажа с такой прямой связью между навыком игрока и выходным уроном.

## Сравнение с другими DPS

| Персонаж | Время разгона | Потолок урона | Зависимость от команды | Навык игрока |
|----------|---------------|---------------|------------------------|--------------|
| Verso | 1-2 хода | Очень высокий | Низкая | Критическая |
| Maelle | Мгновенно (Medalum) | Очень высокий | Средняя | Средняя |
| Sciel | 2-3 хода | Максимальный | Средняя | Средняя |
| Lune | 2-3 хода | Высокий | Низкая | Низкая |

Verso уникален тем, что его потолок урона зависит от навыка игрока, а не от комбинации снаряжения.

## Распространённые ошибки

- **Overload без уверенного парри**: если вы не парируете стабильно — Overload = самоубийство
- **Игнорирование Free Aim**: стрельба — быстрый способ набора ранга между основными атаками
- **End Bringer без Break/Stun**: урон End Bringer по не-оглушённой цели значительно ниже
- **Пренебрежение Agility**: меньше ходов = медленнее набор Perfection = ниже ранг

## Заключение

Verso — персонаж, который вознаграждает мастерство. Чем лучше вы играете — тем сильнее он становится. End Bringer + Perfection Rank S + подготовка командой = числа, от которых боссы исчезают за один ход. Для соло-челленджей и Endless Tower нет лучшего выбора: автономность, масштабирование от навыка и взрывной урон через Overload делают Verso универсальным солдатом Expedition 33.
