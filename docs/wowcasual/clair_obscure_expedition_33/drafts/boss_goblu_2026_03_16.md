# Expedition 33: как победить Goblu

Второй сюжетный босс — и первый, который наказывает пассивную игру. Goblu вводит механику цветов: каждые несколько ходов на арене появляются 4 цветка, и если вы не уничтожите их через Free Aim, босс поглотит их и получит баффы. Один пропущенный раунд цветов — и Goblu начнёт убивать с одного удара. Это бой на скорость, не на выносливость.

## Слабости и данные

| Параметр | Значение |
|----------|----------|
| Слабость | **Молния** |
| Сопротивления | **Лёд** |
| Слабая точка | нет |
| Появления | Flying Waters (обязательный), Endless Tower Stage 3 Trial 3 |
| **XP** | **2 700** |
| **Награды** | Shape of Health, Polished Chroma Catalyst, Chroma Catalyst x3 |

## Механика цветов: подробный разбор

Каждые 2–3 хода Goblu тратит ход на создание **4 цветов** на арене. В этот ход он не атакует — используйте для лечения, если нужно.

**Синие цветы** → при поглощении дают **щиты** (1 щит за цветок). Щиты блокируют прямой урон. Снимаются Free Aim Shot.

**Красные цветы** → при поглощении дают **бафф урона**. Стакается. Два раунда красных баффов → ваншот любого персонажа.

Цветы **иммунны к AoE и навыкам**. Уничтожаются **только Free Aim Shot** — 1 AP за цветок.

**Приоритет уничтожения:** красные > синие. Красные убивают; синие только замедляют.

**Если не успели:** Goblu поглощает все оставшиеся цветы за один ход. Лучше уничтожить 3 из 4, чем ни одного.

### Управление AP на ход с цветами

У вас 4 цветка и 3 персонажа. Каждый Free Aim = 1 AP. Оптимальная раскладка:
- **Персонаж 1:** Free Aim → цветок #1 (красный)
- **Персонаж 2:** Free Aim → цветок #2 (красный)
- **Персонаж 3:** Free Aim → цветок #3 + обычная атака по боссу (если хватает AP)
- Четвёртый цветок (синий) — допустимая жертва. Один синий щит — терпимо.

## Атаки и парирование

### Triple Jump (тройной прыжок задом)

Goblu прыгает **задом вниз** три раза подряд. Каждый прыжок бьёт по всей группе.

Тайминг: парируйте в момент, когда **нижняя часть тела касается земли**. Не раньше — Goblu зависает в воздухе, создавая обманчивую паузу. Между вторым и третьим прыжком пауза чуть длиннее — не парируйте рано.

```
Прыжок 1: зависание → приземление → ПАРИРОВАНИЕ
Прыжок 2: зависание → приземление → ПАРИРОВАНИЕ
Прыжок 3: длинная пауза → приземление → ПАРИРОВАНИЕ
```

Три успешных парирования → **тройная контратака**. Один из лучших источников бесплатного урона в бою.

**Не прыгайте.** Прыжок не уклоняет от этой атаки. Только парирование или додж.

### 2-Hit Melee Combo

Подбегает к случайному персонажу, бьёт правой рукой → левой рукой. Парируйте каждый удар **непосредственно перед контактом**. Умеренный урон — не опасен, если парируете.

### Crushing Attack (при HP <30%)

Goblu подпрыгивает высоко, вращаясь, затем **замедляется в воздухе** и обрушивает оба кулака на одну цель. Потенциально смертельна.

Тайминг: парируйте **в конце фазы замедления**, когда Goblu начинает ускорение вниз. Если не уверены — доджите. Промах парирования здесь может стоить персонажа.

### 4-Hit Melee Combo (при низком HP)

Усиленная версия: четыре удара подряд после поглощения цветов. Ритм равномерный, парируйте каждый. Появляется только после того, как босс поглотил хотя бы одну волну цветов.

## Пошаговая тактика

### Ранний бой (100%–50% HP)

1. **Gustave → Overcharge** (молниевый урон в слабость) каждый ход, когда доступен
2. **Lune → Thunderfall** или молниевые навыки
3. Третий персонаж — Free Aim по цветам + обычные атаки
4. **Парируйте Triple Jump** для тройных контратак
5. При появлении цветов: **все стреляют** красные первыми

### Поздний бой (<50% HP)

1. Цветы появляются чаще — Free Aim становится приоритетнее
2. **Crushing Attack** начинает появляться — будьте готовы к парированию после замедления
3. Если Goblu поглотил красные цветы — **переключитесь на полную защиту** на 1–2 хода, пока бафф не спадёт
4. Не затягивайте — бафф стакается, и каждый раунд цветов делает босса опаснее

## Типичные ошибки

1. **AoE по цветам.** Иммунны к AoE. Только Free Aim. Ход впустую.

2. **Прыжок от Triple Jump.** Не работает. Парирование или додж.

3. **Игнорирование красных цветов.** Один раунд — терпимо. Два раунда красных баффов — ваншот.

4. **Ранний парирование Crushing Attack.** Goblu замедляется в воздухе — это обманка. Ждите начала ускорения вниз.

5. **Бой на истощение.** С каждым раундом цветов босс сильнее. Чем дольше бой — тем хуже. Молниевый урон → быстрое убийство.
