# Expedition 33: как победить Mask Keeper

Mask Keeper — босс, который атакует сразу после победы над Visages, без перерыва на отдых и лечение. Это одно из самых жёстких испытаний второго акта Clair Obscur: Expedition 33, и его сложность напрямую зависит от того, насколько хорошо вы подготовились в предыдущих зонах.

## Слабости и появления

| Параметр | Значение |
|----------|----------|
| Слабость | **Огонь, Тьма** |
| Сопротивления | **Лёд** |
| Появления | Peaks/Visages (обязательный), Endless Tower Stage 11 Trial 1 |
| **XP** | **123 195** |
| **Награды** | Immaculate Picto, Resplendent Chroma Catalyst x6, Recoat x2 |

## Контекст: бой без паузы

Mask Keeper появляется немедленно после победы над Visages. У вас не будет возможности восполнить HP, AP или использовать лагерь. Перед финальным ударом по Visages **вылечите группу максимально**.

## Подготовка: три долины решают всё

Если вы уничтожили Маски Эмоций в Joy Vale, Sadness Vale и Anger Vale — бой с Mask Keeper значительно проще. Каждая маска, уничтоженная в долине, не будет доступна Mask Keeper для баффов и дебаффов. Без масок босс лишается мощных усилений и ослабляющих эффектов, превращаясь в прямолинейного мечника.

Если маски не уничтожены — Mask Keeper получает бафф от каждой выжившей маски и дебаффит вашу группу их эффектами. Бой из сложного становится мучительным.

## Слабости

Mask Keeper уязвим к **Огню** и **Тьме** — тем же стихиям, что и Visages. Удобно: состав и экипировка не требуют перенастройки между боями.

## Паттерны атак

**Storm Attack (штормовая атака).** Серия из четырёх быстрых ударов по одному персонажу. Удары идут в быстрой последовательности — парируйте каждый ритмично. Тайминг: удар-удар-удар-удар с равными интервалами примерно в полсекунды.

**Fire Sword (огненный меч).** Три взмаха мечом, за которыми следует Gradient-атака. Первые три удара парируются стандартно — следите за замахом. Четвёртый (Gradient) требует нажатия кнопки Gradient Counter в момент визуального сигнала. Промах по Gradient-контратаке означает критический урон.

**Chaotic Combo (хаотичное комбо).** Самая сложная атака: восемь ударов и взмахов с разными интервалами. Здесь нет стабильного ритма — каждый удар может прийти быстрее или медленнее предыдущего. Рекомендация: уклоняйтесь, а не парируйте. Окно уклонения прощает больше ошибок, чем парирование в хаотичном комбо.

## Эффекты масок (если не уничтожены)

Выжившие маски усиливают Mask Keeper:
- **Joy** — увеличивает урон босса.
- **Sadness** — снижает вашу защиту.
- **Anger** — даёт боссу дополнительные атаки.

Каждый выживший эффект стакается, делая бой экспоненциально сложнее.

## Рекомендуемый состав и навыки

- **Maelle** — Brulerum + Breaking Burn для огненного урона. Rain of Fire для Burn. Combustion для взрывного DPS.
- **Sciel** — Grim Harvest для тёмного урона. Phantom Blade в состоянии Twilight.
- **Lune** — Fire Rage для пассивного огненного урона каждый ход. Wildfire для Burn.

Альтернатива: Verso для очистки дебаффов, если маски не были уничтожены.

## Стратегия боя

1. **Открытие:** Maelle — Rain of Fire для Burn. Sciel — Grim Harvest для тёмного урона. Lune — Fire Rage для постоянного огненного урона.
2. **Середина боя:** поддерживайте Burn, чередуйте огненный и тёмный урон. Free Aim Shot в слабые точки для дополнительного урона.
3. **Chaotic Combo:** переключайтесь на уклонение. Не пытайтесь парировать все восемь ударов — одна ошибка может стоить персонажа.
4. **Gradient-контратаки:** всегда будьте готовы к Fire Sword. Gradient Counter наносит огромный урон — не пропускайте эти окна.

## Советы

- Лечите группу перед последним ударом по Visages — это ваша единственная возможность войти в бой с полным HP.
- Три долины — обязательны. Без уничтоженных масок бой на порядок сложнее.
- Chaotic Combo — главная угроза. Уклонение безопаснее парирования.
- Экономьте Gradient Charge для Fire Sword контратак — они наносят максимальный урон.
- Если ресурсы на исходе — играйте агрессивно. Затяжной бой без лечения — верная смерть.
