# Expedition 33: как победить Dualliste

Первый босс второго акта — и мгновенный скачок сложности. Dualliste сражается в двух стойках (один меч → два меча), вводит статус **Inverted** (исцеление = урон по себе), а его комбинации требуют чередования парирования и прыжков в конкретной последовательности. Если Eveque и Goblu проверяли основы, то Dualliste проверяет, насколько вы их усвоили.

## Слабости и данные

| Параметр | Значение |
|----------|----------|
| Слабость | **Свет (Light)** |
| Сопротивления | **Огонь, Тьма** |
| Появления | Forgotten Battlefield (обяз.), Flying Manor, Endless Tower (Stage 1/11) |
| **XP** | **27 560** |
| **Награды** | Dualiso (оружие), Combo Attack I (навык), Polished Chroma Catalyst x3, **3 950 Chroma** |

**Огонь и Тьма бесполезны.** Maelle и Sciel — плохой выбор для урона. **Verso** со световыми навыками (Perfect Break, Purification, Follow Up) — ваш главный дамагер.

## Фаза 1: Один меч (100%–~50% HP)

Dualliste сражается одним горящим мечом. Комбинации длинные, но с чёткими визуальными сигналами.

### 5-Hit Combo (пятиударная серия)

Подходит к цели, отводит правую руку назад, затем:
```
ЛЕВО → ЛЕВО → ПРАВО → ПРАВО → СВЕРХУ (над головой)
```
Парируйте каждый удар. Тайминг равномерный — нет пауз-обманок. Финальный удар сверху чуть медленнее — не торопитесь.

### Swiping Combo (комбо с прыжком)

Четыре удара с чередованием парирования и прыжка:
```
1: ПАРИРОВАНИЕ (удар слева)
2: ПРЫЖОК (восходящий удар снизу-вверх)
3: ПАРИРОВАНИЕ (удар сверху)
4: ПАРИРОВАНИЕ (прыжковый удар с высоты)
```
Ключевой момент — **второй удар, прыжок**. Он идёт снизу вверх, парирование не сработает. Запомните последовательность: пар-прыг-пар-пар.

### AoE Expedition Attack

Раскручивает меч, втыкает в землю — огненные волны расходятся по земле по всей группе. Парируйте в момент втыкания меча в землю.

### Swift Attack (3-Hit)

Поднимает меч над головой, затем три удара: лево → сверху → право. Пауза в воздухе между вторым и третьим — не парируйте рано.

## Переход фазы: кат-сцена

При ~50% HP — кат-сцена. Dualliste достаёт второй меч. **Статус-эффекты из первой фазы сохраняются.** Если вы наложили Burn или Mark — они переносятся.

## Фаза 2: Два меча (~50%–0% HP)

Все атаки первой фазы остаются, но усиливаются. Плюс новые.

### Extended Swift Combo (6 ударов)

Шесть ударов с чередованием:
```
1: ПАРИРОВАНИЕ
2: ПРЫЖОК
3: ПРЫЖОК
4: ПАРИРОВАНИЕ
5: ПАРИРОВАНИЕ
6: ПАРИРОВАНИЕ
```
Самая сложная серия в бою. Удары 2–3 бьют по нижней части тела — только прыжок. Удары 4–6 идут быстро подряд — три парирования без пауз. Выучите этот ритм: **пар-прыг-прыг-пар-пар-пар**.

### AoE Expedition Attack (две волны)

Теперь два втыкания → две волны огня. Парируйте оба. Вторая волна плохо видна из-за яркости первой — полагайтесь на звук, не на визуал.

### Stormblood

Три рывка в разных направлениях + финальный укол. Парируйте каждый рывок и укол. Направления меняются — следите за движением тела, не за мечом.

### Inverted (инверсия лечения)

В середине второй фазы Dualliste втыкает оба меча в землю и накладывает **Inverted** на всю группу.

**Inverted превращает исцеление в урон.** Бустеры, регенерация, Healing Light — всё наносит урон вместо лечения. Regen-эффекты тоже конвертируются.

Что делать:
1. **Не лечитесь.** Любое исцеление убьёт вас.
2. **Chroma Elixir** снимает Inverted — используйте, если нужно срочно вылечиться.
3. **Переключитесь на полную атаку.** Цель — убить Dualliste до того, как отсутствие лечения вас добьёт.
4. **Maelle → защитная стойка**, если кто-то на критическом HP — пусть танкует, пока остальные добивают.

## Пошаговая тактика

### Фаза 1

1. **Verso** — Perfect Break, Purification (световой урон в слабость)
2. **Lune** — Healing Light для поддержки, световые навыки для урона
3. Парируйте всё. Заучите: пар-пар-пар-пар-пар (5-Hit), пар-прыг-пар-пар (Swiping)
4. Наложите Burn и Mark перед переходом — они перенесутся

### Фаза 2

1. Агрессивный урон через Verso
2. Заучите новый ритм: пар-прыг-прыг-пар-пар-пар (Extended Swift)
3. При Inverted — **не лечитесь, добивайте**. Chroma Elixir если критично
4. При ~25% HP — выкладывайте всё, не экономьте

## Типичные ошибки

1. **Огненные/тёмные навыки.** Босс сопротивляется обоим. Только Свет.

2. **Парирование вместо прыжка на ударах по ногам.** Swiping Combo удар #2 — прыжок. Extended Swift удары #2–3 — прыжок. Парирование здесь не сработает.

3. **Лечение при Inverted.** Самая фатальная ошибка. Лечение = урон по себе. Снимите Inverted Chroma Elixir или терпите.

4. **Раннее парирование AoE второй волны.** Вторая волна едва видна из-за яркости. Слушайте звук втыкания меча.

## Награды

**Dualiso** — оружие. **Combo Attack I** — навык, открывающий новые комбинации. Оба полезны для дальнейшего прохождения второго акта.
