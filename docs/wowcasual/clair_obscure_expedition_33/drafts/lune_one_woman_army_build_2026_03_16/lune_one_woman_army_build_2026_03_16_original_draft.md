# Lune: One-Woman Army — Stains, Esquim и DLC-билд

Lune — маг Expedition 33. Элементальные заклинания, лечение, дебаффы и, при правильной сборке, способность уничтожить целую группу врагов за один ход. Её механика Elemental Stains превращает каждое заклинание в часть нарастающей цепной реакции. Разберём все её билды: от раннего саппорта до One-Turn Army в эндгейме.

## Механика Elemental Stains

### Как работают Stains

Каждый раз, когда Lune использует заклинание определённого элемента, на ней накапливаются Elemental Stains (элементальные пятна) этого элемента. Стаки Stains усиливают последующие заклинания того же элемента и разблокируют бонусные эффекты.

### Стратегия Stains

- Последовательное использование одного элемента = нарастающая мощь
- Переключение между элементами сбрасывает стаки (или не даёт накопить)
- Некоторые оружия дают бонусы за чередование элементов — баланс зависит от билда

## Каталог оружий по актам

### Акт 1: Trebuchim

**Скейлинг**: Vitality и Luck

**Пассив (Level 4)**: усиливает базовые заклинания

**Роль**: Trebuchim идеален для раннего саппорта. Скейлинг с Vitality даёт Lune выживаемость, а Luck — шанс на крит для её заклинаний. В Акте 1 Lune лучше всего работает как хилер с дополнительным уроном, поскольку у команды ещё нет ресурсов для чистого DPS-билда.

**Билд с Trebuchim**:
- Приоритет: Vitality > Luck > Agility
- Умения: лечение + элементальные дебаффы
- Роль: хилер-саппорт

### Акт 2: Kralim

**Скейлинг**: зависит от чередования элементов

**Ключевой пассив**: увеличивает урон, если каждое последующее заклинание — другого элемента. Дополнительно: даёт бесплатные Stains, если у Lune их нет.

**Билд с Kralim**:
- Приоритет: Luck > Agility > Defence
- Стратегия: чередуйте элементы каждый ход — огонь → лёд → молния → огонь
- Пассив Kralim компенсирует потерю стаков от чередования, давая бесплатные Stains
- Роль: гибрид DPS/саппорт

### Акт 3 / Эндгейм: Choralim

**Скейлинг**: Defence и Agility

**Пассив**: усиливает элементальные комбо при высоких стаках Stains

**Билд с Choralim**:
- Приоритет: Defence > Agility > Luck
- Стратегия: фокус на одном элементе для максимальных стаков Stains
- Пассив Choralim масштабируется с Defence, создавая синергию между защитой и уроном
- Роль: чистый DPS

### DLC (Verso's Drafts): Esquim

**Источник**: DLC Verso's Drafts (Thank You Update, декабрь 2025)

**Ключевые особенности**: Esquim — новое оружие для Lune из DLC, расширяющее её арсенал новыми пассивами и механиками.

**Билд с Esquim**:
- Новые Pictos из DLC усиливают элементальные комбо
- Комбинация с DLC-Luminas для максимизации Stains
- Роль: финальный DPS-билд для пост-эндгейм контента

## Каталог билдов

### Билд 1: Early Healer (Акт 1)

**Концепция**: Lune — единственный надёжный хилер в ранней игре. Используйте это.

**Оружие**: Trebuchim
**Атрибуты**: Vitality (max) > Luck > Agility
**Умения**: все лечебные + один элементальный для дебаффа
**Pictos**: на усиление хила, на регенерацию

**Когда переходить**: как только получите Kralim и достаточно ресурсов для DPS-пикосов.

### Билд 2: Element Juggler (Акт 2)

**Концепция**: чередование элементов каждый ход для активации пассива Kralim.

**Оружие**: Kralim
**Атрибуты**: Luck > Agility > Defence
**Умения**: по одному заклинанию каждого элемента + одно лечебное
**Pictos**: бонус урона при смене элемента, крит-пикосы

**Ротация**:
1. Огонь (+ бесплатные Stains от Kralim)
2. Лёд (+ бонус от смены элемента)
3. Молния (+ бонус)
4. Хил (если нужен) или огонь (цикл заново)

### Билд 3: Archmage DPS (Акт 3)

**Концепция**: максимальные стаки одного элемента + мощный финишер.

**Оружие**: Choralim
**Атрибуты**: Defence > Agility > Luck
**Умения**: 3 заклинания одного элемента + финишер
**Pictos**: максимум стаков Stains, крит при полных стаках

**Ротация**:
1. Элементальное заклинание 1 → стаки Stains
2. Элементальное заклинание 2 → больше стаков
3. Финишер → расход стаков с максимальным бонусом

### Билд 4: One-Turn Nuke (Эндгейм)

**Концепция**: уничтожение группы врагов или босса за один ход.

**Оружие**: Choralim или Esquim (DLC)
**Атрибуты**: Defence > Luck > Agility
**Умения**: максимально урон-заклинания + AoE
**Pictos**: максимальный крит, бонус при полном HP, бонус от стаков

**Setup** (от команды):
1. Густав: Marking Shot на основную цель
2. Verso: Break на основную цель
3. Lune: серия заклинаний с полными стаками → One-Turn Kill

## Дополнительный контент из DLC Verso's Drafts

### Новое оружие Esquim

Esquim расширяет возможности Lune в пост-эндгейме. Точные характеристики зависят от уровня прокачки, но основное направление — усиление элементальных комбо.

### Новые Pictos

DLC добавляет 17 новых Pictos, часть из которых напрямую усиливает элементальные билды Lune:
- Longer Break — продлённое оглушение для больше времени на стаки
- Double Third — удвоение определённых бонусов

### Новые боссы

Боссы DLC требуют адаптации билдов. Некоторые имеют элементальные резисты, вынуждающие Lune переключать стратегию на ходу.

## Распространённые ошибки

- **DPS Lune в Акте 1**: у вас нет ресурсов для этого. Хильте.
- **Один элемент с Kralim**: Kralim даёт бонус за смену элементов. Один элемент — потеря пассива.
- **Один элемент без Kralim**: без Choralim или Esquim фокус на одном элементе неэффективен.
- **Игнорирование Stains**: следите за индикатором стаков. Без стаков финишер бьёт слабо.
- **Забывание лечить**: даже в DPS-билде Lune — запасной хилер. Не дайте команде умереть ради одного лишнего заклинания.

## Заключение

Lune — один из самых гибких персонажей Expedition 33. От хилера в Акте 1 до One-Turn Nuke в эндгейме — её путь требует смены оружия и стратегии каждый акт. Ключ к мастерству — понимание системы Stains и правильный выбор между накоплением стаков и их расходом. С DLC-оружием Esquim и новыми Pictos её потолок урона поднимается ещё выше.
