# Expedition 33: как победить Renoir (финальный бой)

Финальный бой всей игры. Две фазы, семь атак, механика Vanish (навсегда убирает персонажа из боя), призыв Visages для щитов и скриптовый финал от Maelle. Рекомендуемый уровень — 70+. Это самый длинный и насыщенный бой в Expedition 33.

## Данные

| Параметр | Значение |
|----------|----------|
| Слабость | **Нет** |
| Рек. уровень | **70+** |
| Появления | Lumiere (финальный) |
| **XP** | **183 000** |

## Основная стратегия урона: Burn + Virtuose

Нет стихийных слабостей, но есть мощная связка:

1. **Lune → Wildfire** или **Maelle → Rain of Fire** — наложите **Burn** на Renoir
2. **Maelle → Swift Stride** — переключает Maelle в **Virtuose Stance** (требует горящую цель)
3. **Maelle → Fleuret Fury** в Virtuose Stance → **200% урона**

Поддерживайте Burn весь бой. Каждый ход без Burn — потерянный множитель урона.

## Атаки фазы 1

### Void Cross (крест пустоты)

Горизонтальный рассекающий удар мечом. Звуковой сигнал предшествует удару. Парируйте **после звукового щелчка**, до попадания.

### Black Hole (чёрная дыра)

Формируется ~5 секунд, затем взрывается. Парируйте/доджите в момент взрыва. Долгая подготовка даёт время подготовиться.

### Slow Swing (медленный замах)

Круговое движение мечом. **Единственная атака, требующая ПРЫЖОК, а не парирование.** Запомните: медленный, круговой = прыжок.

### Five-Hit Combo (5 ударов)

```
Удар 1-4: БЫСТРО подряд → ПАРИРОВАНИЕ каждого
Удар 5: ПАУЗА → удар → ПАРИРОВАНИЕ
```
Подвох — пятый удар задержан. Не парируйте его в ритме первых четырёх.

### Void Meteors (метеоры пустоты)

Пять метеоров падают один за другим. Парируйте каждый ритмично — интервал равномерный. Пять парирований = пять контратак.

### Six-Hit Combo (6 ударов)

```
Удар 1: одиночный → ПАРИРОВАНИЕ
Удары 2-6: БЫСТРО подряд (5 штук) → ПАРИРОВАНИЕ каждого
Удар 7 (финал): ПАУЗА → удар → ПАРИРОВАНИЕ
```
Похож на Five-Hit, но с дополнительным стартовым ударом. Финальный — задержан.

### Summoning Axons (призыв аксонов)

Аксоны атакуют **мгновенно при появлении**. Реагируйте быстро — Free Aim по ним или парирование их ударов. Не игнорируйте.

### Vanish (исчезновение) — Gradient Counter обязателен

Renoir создаёт сферу пустоты и направляет её в персонажа. Если не выполнить **Gradient Counter** — цель **навсегда исчезает из боя**. Не умирает — исчезает. Воскрешение невозможно.

Тайминг: нажмите Gradient, когда Renoir **замахивается для удара по сфере**. Визуальный сигнал чёткий — замах рукой назад. Не торопитесь.

**Это абсолютный приоритет.** Всё остальное вторично. Потеря персонажа навсегда — катастрофа для DPS и лечения.

### Shield от Visages

Renoir призывает Visages для создания щитов. Снимайте **Free Aim Shot** или **Breaking Rules** (Maelle).

## Фаза 2 (~50% HP)

HP значительно ниже, но урон возрастает. Атаки те же, но быстрее. Один пропущенный удар может убить.

**Скриптовый финал:** при критически низком HP Renoir готовит финальную атаку. **Maelle автоматически** контрит её через кат-сцену, завершая бой. Не нужно ничего нажимать — просто доведите HP до порога.

## Пошаговая тактика

### Фаза 1

1. **Burn** на Renoir → Maelle → Swift Stride → Virtuose → Fleuret Fury (200% урона)
2. Парируйте всё. Помните: Slow Swing = ПРЫЖОК, остальное = парирование
3. **Gradient Counter на Vanish** — абсолютный приоритет, без исключений
4. Free Aim по щитам от Visages
5. Аксоны — убивайте сразу

### Фаза 2

1. Продолжайте Burn + Virtuose связку
2. Атаки быстрее — парирование сложнее, но паттерны те же
3. Не экономьте ничего — это последний бой
4. Доведите HP до порога → скриптовый финал Maelle

## Типичные ошибки

1. **Парирование Slow Swing.** Только прыжок. Круговой замах = прыжок.

2. **Пропуск Gradient Counter на Vanish.** Персонаж исчезает навсегда. Ничто не важнее этого парирования.

3. **Ранний парирование пятого/седьмого удара в комбо.** Задержан. Не парируйте в ритме предыдущих.

4. **Нет Burn на боссе.** Virtuose Stance (200% урона) требует горящую цель. Без Burn — половина урона.

5. **Экономия ресурсов.** Это финал. Тратьте всё.
