# Expedition 33: как победить Glissando

Гигантский червь в полосах ткани, двигающийся как змея. Glissando обитает в зоне Sirene и вводит две ключевые механики: **Charm** (очарование, контролирующее ваших персонажей) и **поглощение павших** — если персонаж гибнет, Glissando пожирает тело и восстанавливает HP. Каждая смерть в вашей группе — лечение для босса.

## Слабости и данные

| Параметр | Значение |
|----------|----------|
| Слабость | **Лёд, Тьма** |
| Сопротивления | **Земля, Свет** |
| Появления | Sirene (обязательный) |
| **Награды** | Sirene Outfit (Lune), Resplendent Chroma Catalyst, **2 820 Chroma** |

## Механика хвоста и Charm

Хвост Glissando — ваш тактический рычаг. **Free Aim Shot по хвосту (погремушке):**
- Ослабляет босса
- **Снимает Charm** с очарованного персонажа

Charm заставляет вашего персонажа атаковать союзников. Если кого-то очаровали — немедленно стреляйте в хвост. Это приоритетнее урона по боссу.

## Атаки и парирование

### Head Smash (удар головой)

Glissando поднимает тело, затем бьёт головой дважды:
```
1-й удар: ЗАМЕДЛЕННЫЙ (босс поднимается → пауза → удар) → ПАРИРОВАНИЕ
2-й удар: МГНОВЕННЫЙ (сразу после первого) → ПАРИРОВАНИЕ
```

Подвох — **второй удар почти мгновенный** после первого. Парируйте первый, когда босс перестаёт подниматься. Второй — немедленно после первого, без паузы.

### Summon Ballets (призыв танцоров)

Glissando вызывает **3 Балета** — вращающихся существ, атакующих группу:
```
Балет #1: справа, взмах снизу-вверх → ПРЫЖОК
Балет #2: слева, взмах снизу-вверх → ПРЫЖОК
Балет #3: справа, взмах снизу-вверх → ПРЫЖОК
```

Прыгайте **после того, как Балет прекратил вращение** и начал взмах. Каждый успешный прыжок → контратака.

### Tail Swipe (удар хвостом)

Хвост раскачивается и бьёт по группе. Парируйте или доджите. Менее опасен, чем Head Smash.

## Пошаговая тактика

1. **Lune** → ледяные атаки (Immolation, Ice Lance) для урона в слабость + Burn/Freeze
2. **Sciel** → тёмные атаки для дополнительного урона в слабость
3. При Charm → **Free Aim в хвост немедленно**, не ждите своего хода
4. Прыгайте через всех трёх Балетов → три контратаки бесплатно
5. Head Smash: первый парируйте после подъёма, второй — мгновенно после первого

## Типичные ошибки

1. **Не стреляете в хвост при Charm.** Очарованный персонаж бьёт союзников — снимите Charm как можно быстрее.

2. **Ранний прыжок от Балетов.** Ждите окончания вращения. Прыжок во время вращения не сработает.

3. **Позволили персонажу умереть.** Glissando поглощает тело и восстанавливает HP. Каждая смерть — шаг назад. Держите HP группы высоким.

4. **Земляные/световые атаки.** Босс сопротивляется обоим. Только Лёд и Тьма.
