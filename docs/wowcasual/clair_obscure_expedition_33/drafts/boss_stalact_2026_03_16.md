# Expedition 33: как победить Stalact

Stalact — босс с двумя стихийными стойками, тремя ритмичными контратаками за один ход и механикой перегрева, которая убивает группу за два хода, если не снести шесть щитов. Бой-учебник: здесь вы впервые используете Gradient-атаки и учитесь чередовать стихии на лету.

## Слабости и появления

| Параметр | Значение |
|----------|----------|
| Слабость | **Огонь** (в Ледяной стойке) / **Лёд** (в Огненной стойке) |
| Поглощает | Урон той же стихии, что и текущая стойка |
| Появления | Monoco's Station (обязательный), Flying Manor |

**Критически важно:** урон стихией текущей стойки **лечит** босса. Попали Огнём по Огненному Stalact — восстановили ему HP. Стойка меняется только от **навыков** противоположной стихии — обычные атаки, Free Aim и контратаки стойку не переключают.

## Двойная стихийная механика: ритм боя

```
Ледяная стойка → бейте ОГНЁМ → босс переключается на Огненную
Огненная стойка → бейте ЛЬДОМ → босс переключается на Ледяную
→ повтор
```

Lune — MVP. К этому моменту у неё должно быть минимум два огненных и два ледяных заклинания. Чередуйте их по стойке босса.

## Атаки и парирование

### Earthquake (землетрясение) — три удара, три контратаки

Stalact бьёт по земле последовательно:
```
1: ПРАВЫЙ КУЛАК → прыжок → КОНТРАТАКА
2: ЛЕВЫЙ КУЛАК → прыжок → КОНТРАТАКА
3: ОБА КУЛАКА (прыжок + удар) → прыжок → КОНТРАТАКА
```

Первые два удара идут **близко друг к другу**. Третий — после **заметной паузы**. Не прыгайте рано на третий — задержка обманывает.

**Это один из лучших источников контратак в игре.** Три прыжка = три контратаки за один ход босса. Если вы освоили тайминг — Earthquake становится вашим лучшим другом, а не угрозой.

### Single-Target Combo (4 удара по одной цели)

Stalact подходит к одному персонажу и бьёт кулаками:
```
ПРАВЫЙ → ЛЕВЫЙ → ПРАВЫЙ → ЛЕВЫЙ
```

Каждый удар имеет ветровой замах — парируйте, когда кулак начинает движение вниз. Ритм быстрый, но равномерный — без пауз и обманок. Додж тоже работает.

### Heating Up (перегрев) — таймер на 2 хода

Stalact объявляет «heating up», тело светится оранжевым, и он накладывает **6 щитов**. Через **2 хода** — взрыв, наносящий экстремальный урон по всей группе. Фактически вайп.

**Что делать:**
1. **Снесите все 6 щитов за 2 хода.** Каждый удар снимает 1 щит.
2. **Мульти-хит навыки:** Gustave → Overcharge, Maelle → Breaking Rules — снимают несколько щитов за ход.
3. **Стихия не важна** при снятии щитов — бейте чем угодно, лишь бы мульти-хит.
4. Если не успели — пробуйте парировать/доджить взрыв. Тайминг сложный, но возможный.

## Gradient-атаки (обучение Монако)

В этом бою Монако объясняет Gradient-систему:
- За каждый 1 AP, потраченный на навыки, шкала Gradient заполняется на **5%**
- При 100% получаете **Gradient Charge** (максимум 3 заряда)
- Gradient-атака даёт **автоматический второй ход** — фактически удвоение урона

**Когда использовать Gradient:** сразу после смены стойки, когда босс уязвим к вашей стихии. Gradient-атака + обычный ход = два полных хода урона в слабость.

## Пошаговая тактика

### Основной ритм

1. Определите стойку (визуально: ледяная = синий, огненная = красный)
2. Бейте противоположной стихией через **навыки** (не Free Aim — он не меняет стойку)
3. Стойка сменилась → переключите стихию
4. Прыгайте через Earthquake для тройных контратак
5. При Heating Up — мульти-хит навыки, стихия не важна

### Экономия AP

Earthquake даёт 3 контратаки бесплатно — используйте это для экономии AP. На своём ходу тратьте AP на стихийные навыки и Gradient, а контратаки из Earthquake дают «бесплатный» урон.

## Типичные ошибки

1. **Бьёте стихией текущей стойки.** Лечите босса. Всегда противоположная стихия.

2. **Free Aim для смены стойки.** Не работает. Стойка меняется только от навыков.

3. **Ранний прыжок на третьем ударе Earthquake.** Пауза перед третьим ударом длиннее. Ждите.

4. **Паника при Heating Up.** 6 щитов за 2 хода — сложно, но возможно. Overcharge + Breaking Rules = 5+ щитов за ход.

5. **Экономите Gradient.** Gradient даёт второй ход. Используйте его в момент уязвимости босса, не копите.
