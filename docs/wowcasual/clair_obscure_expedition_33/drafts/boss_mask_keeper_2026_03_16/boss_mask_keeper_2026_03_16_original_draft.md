# Expedition 33: как победить Mask Keeper

Бой без паузы после Visages, шесть различных комбинаций с разными таймингами, механика масок из трёх долин и статус Bound, блокирующий действия. Mask Keeper — один из самых тактически насыщенных боёв во втором акте.

## Слабости и данные

| Параметр | Значение |
|----------|----------|
| Слабость | **Огонь, Тьма** |
| Сопротивления | **Лёд** |
| Появления | Peaks/Visages (обязательный), Endless Tower Stage 11 |
| **XP** | **123 195** |
| **Награды** | Immaculate Picto, Resplendent Chroma Catalyst x6, Recoat x2 |

## Подготовка: три долины = три отключённых маски

Перед боем пройдите **Joy Vale, Sadness Vale, Anger Vale** и уничтожьте маски в каждой. Каждая убитая маска — одна способность, которую Mask Keeper **не сможет использовать**. Без масок он лишается баффов и дебаффов, превращаясь в прямолинейного мечника.

**Бой начинается сразу после Visages — без перерыва.** Перед финальным ударом по Visages вылечите группу максимально. HP, AP и расходники не восполняются.

## Атаки и парирование

### Expedition Attack (базовая волна)

Вращает оружие, затем:
```
ВЫПАД → ВОЛНА #1 → ВОЛНА #2 → пауза → ВОЛНА #3 → ВОЛНА #4
```
Парируйте каждую волну. Визуальный сигнал: вращение оружия → волна. Между второй и третьей — **пауза** — не парируйте рано.

### Vehement Attack (усиленная волна, 6 повторов)

Та же механика, что Expedition Attack, но **повторяется 6 раз**. На последнем ударе накладывает **Bound** (персонаж не может действовать 1 ход).

Парирование идентично — но 6 раз подряд. Концентрация и ритм. Промах на последнем ударе = Bound на ключевом персонаже.

### Chaotic Combo (8 ударов по одной цели)

Подходит к одной цели и бьёт 8 раз: замахи, вращения, финальный выпад.
```
Парируйте каждый удар, когда ВИДИТЕ ЗАМАХ МЕЧА к цели.
```
Ритм неравномерный — некоторые удары быстрые подряд, другие с паузой. Не полагайтесь на ритм — следите за мечом визуально.

### Long Swing Combo (6 ударов, переменный ритм)

```
Удар 1-2: БЫСТРО подряд (почти одновременно)
Удар 3: ПАУЗА → удар
Удар 4-5: БЫСТРО подряд
Удар 6: ПАУЗА → удар
```
Ритм: быстро-быстро-пауза-быстро-быстро-пауза. Запомните этот шаблон — он повторяется каждый раз идентично.

### Fire Sword (огненный меч + Gradient Counter)

Три взмаха огненным мечом, затем **Gradient Attack**. Первые три парируйте обычно. На Gradient — используйте **Gradient Counter** (кнопка Gradient, не парирование). Промах Gradient Counter = огромный урон.

### Ice Sword (ледяной меч + Freeze)

Четыре ледяных волны из меча. Могут наложить **Freeze**. Парируйте, когда белая энергия собирается на рукояти меча и начинает двигаться к вам.

## Пошаговая тактика

1. **Maelle** → огненные навыки (Rain of Fire, Spark) + Burn DoT
2. **Sciel** → тёмные навыки + Foretell для усиления
3. Третий персонаж — лечение и поддержка
4. **Парируйте всё.** Каждая атака парируемая. Контратаки — основной источник урона помимо навыков.
5. При Fire Sword → готовьте Gradient Counter на финальном ударе
6. При Bound → лечите связанного, ждите следующий ход

## Типичные ошибки

1. **Не прошли три долины.** С активными масками бой значительно сложнее. Маски дают боссу дополнительные баффы и дебаффы на группу.

2. **Не вылечились перед финишем Visages.** Бой начинается мгновенно. Без HP вы не переживёте первую комбинацию.

3. **Ранний парирование в Long Swing.** Ритм быстро-быстро-пауза. Третий и шестой удары задержаны.

4. **Парирование вместо Gradient Counter на Fire Sword.** Финальный удар — Gradient Attack. Обычное парирование не сработает.

5. **Ледяные навыки.** Босс сопротивляется Льду. Только Огонь и Тьма.
