# Expedition 33: как победить Francois

Черепаха с одной атакой. Одной. Но эта атака мгновенно убивает любого персонажа с полного здоровья, если вы не парируете или не доджите. Francois — босс-головоломка, который проверяет ровно один навык: умение распознать визуальный сигнал и нажать кнопку в правильный момент. Освоили ритм — бой тривиален. Не освоили — бесконечный цикл смертей.

## Слабости и данные

| Параметр | Значение |
|----------|----------|
| Слабость | **Тьма (Dark)** |
| Слабая точка | нет |
| Появления | Esquie's Nest (обязательный), Endless Tower Stage 8 Trial 1 |
| **XP** | **1 857** |
| **Награды** | Augmented First Strike (Picto), **392 Chroma** |

## Пять щитов — ваша первая задача

Francois начинает бой с **пятью щитами**. Каждый блокирует одну атаку, вне зависимости от её силы — слабый удар снимает щит так же, как и ультимейт. Это значит:

**Не тратьте мощные навыки на щиты.** Один слабый удар = один щит. Используйте базовые атаки (1 AP) для экономии ресурсов.

Быстрые способы снятия:
- **Maelle → Breaking Rules** — мульти-хит навык, снимает несколько щитов за один ход
- **Lune → Wildfire** — накладывает Burn, который снимает щиты в конце хода (пассивная чистка)
- **Gustave → Lumiere Assault** — до 5 ударов, но лучше приберечь для чего-то полезнее

После снятия всех щитов переключайтесь на тёмный урон через Sciel.

## Единственная атака: Strongest Ice Attack Ever

Francois собирает ледяные частицы из окружения — это визуальный сигнал начала. Затем:

```
Зарядка → ВСПЫШКА #1 → пауза → ВСПЫШКА #2 → ЛУЧ
```

**Тайминг парирования:** нажимайте **сразу после второй вспышки**, когда луч взрывается ослепительной белой вспышкой. Не на первой — на **второй**.

**Если парировали:** контратака + нулевой урон. Каждый успешный парирование — бесплатный ход урона.

**Если не парировали:** один из ваших персонажей мёртв. Луч наносит достаточно урона, чтобы убить с полного HP на стандартном уровне.

**Альтернатива — додж:** окно шире, чем у парирования. Если не уверены в тайминге — доджите. Вы теряете контратаку, но сохраняете персонажа.

### Как тренировать тайминг

Первые 2–3 атаки Francois — используйте для тренировки. Доджите первую, попробуйте парировать вторую. Если промахнулись — воскресите и пробуйте снова. Визуальный ритм запоминается быстро:

Главная ошибка — реакция на **первую** вспышку. Первая вспышка — обманка. Вторая — сигнал к парированию.

## Пошаговая тактика

### Ходы 1–3: Снятие щитов

1. Maelle → Breaking Rules (снимает 2–3 щита)
2. Lune → Wildfire (Burn снимает ещё 1 щит в конце хода)
3. Базовые атаки остальных — по одному щиту за удар
4. **Парируйте/доджите Strongest Ice Attack** каждый ход босса

### Ходы 4+: Урон по голому боссу

1. Sciel → тёмные атаки (бьют в слабость + Foretell для усиления)
2. Maelle → Percee (если есть оружие из Gestral Arena — один Percee снимает >50% HP)
3. Продолжайте парировать каждый ход

Francois **не добавляет новых атак** при низком HP. Ледяной луч — всё, что у него есть. Бой становится монотонным: парируйте → бейте → парируйте → бейте.

## Рекомендуемый состав

- **Sciel** — основной дамагер, тёмный урон + Foretell
- **Maelle** — Breaking Rules для щитов, затем Percee для бурста
- **Lune** — Wildfire для Burn (пассивное снятие щитов), затем поддержка

## Типичные ошибки

1. **Парирование на первой вспышке.** Первая — обманка. Вторая — сигнал. Запомните: два «бипа», парирование на втором.

2. **Мощные навыки по щитам.** Один удар = один щит. Не тратьте 3-AP ультимейты на то, что снимает базовая атака за 1 AP.

3. **Забыли воскресить убитого.** Следующий луч может попасть в того же персонажа или в нового. Воскрешайте немедленно.

4. **Не использовали Burn.** Wildfire снимает щиты пассивно — один бесплатный щит за ход без траты AP.
