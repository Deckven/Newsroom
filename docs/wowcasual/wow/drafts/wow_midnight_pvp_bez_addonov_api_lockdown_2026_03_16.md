# PvP без аддонов: как API Lockdown меняет арену

Патч 12.0 принёс самое контроверзное изменение за годы: Blizzard ограничила доступ аддонов к боевым данным в реальном времени. Combat log стал «чёрным ящиком» — аддоны видят, что событие произошло, но не могут прочитать детали для принятия решений. Для PvP-игроков, годами полагавшихся на OmniBar, GladiusEx и десятки утилит, это землетрясение.

## Что конкретно изменилось

### API-ограничения
Реальные боевые данные перемещены в категорию «секретных значений». Аддоны по-прежнему получают уведомления о событиях (каст начат, бафф применён), но числовые и идентификационные данные скрыты. Это значит:

- Аддоны **не могут** автоматически определять, какую именно способность кастует противник
- Аддоны **не могут** рассчитывать урон входящих способностей
- Аддоны **не могут** точно отслеживать кулдауны противника через пассивный мониторинг лога

### Что затронуто
- **OmniBar** — сильно ограничен. Трекинг вражеских кулдаунов через combat log больше невозможен в прежнем виде
- **Автоматические предупреждения** — аддоны не могут генерировать контекстные алерты на основе вражеских действий
- **DPS-метры в реальном времени** — работают с задержкой или неточно в PvP

### Что выжило
- **Gladius** — по-прежнему работает как стандартный аренный фрейм. Трекинг тринкетов, DR, кастбары, здоровье и мана — всё функционирует
- **DR-трекинг** — вражеский Diminishing Returns отслеживается через лог боя, не через секретные значения
- **Кастомизация фреймов** — визуальная настройка аренных фреймов не затронута

## Встроенные инструменты Blizzard

Blizzard не просто убрала аддоны — она добавила нативные замены:

### DR Tracker
Путь: Options > Gameplay > Diminishing Returns. Встроенный трекер отображает текущие DR-стаки на вражеских целях прямо на фреймах. Не требует аддонов.

### Звуковые оповещения
Нативные звуковые алерты при вражеских кастах. Настраиваются через Options > Sound > Combat. Заменяют функционал, который ранее обеспечивали аддоны вроде OmniBar.

### Улучшенные аренные фреймы
Стандартные аренные фреймы теперь показывают:
- Внешние дефензивные кулдауны (Pain Suppression, Guardian Spirit)
- Статус тринкета
- Иммунитеты
- Всё это — без аддонов

## Как адаптироваться: пошаговый план

### Шаг 1. Аудит аддонов
Проверьте каждый PvP-аддон. Какие функции работают, какие сломаны? Удалите неработающие — они засоряют интерфейс и могут вызывать ошибки.

**Оставить:**
- Gladius/GladiusEx (фреймы)
- OmniCD (частично работает для группового трекинга)
- BigDebuffs (отображение CC на фреймах)

**Удалить или заменить:**
- OmniBar (ограничен критически)
- Любые аддоны автоматического оповещения о вражеских кастах

### Шаг 2. Настройка нативного UI
1. Включите DR Tracker: Options > Gameplay > Diminishing Returns > Show on Target/Arena Frames
2. Настройте звуковые алерты: Options > Sound > Combat Alerts > Enable
3. Проверьте аренные фреймы: Options > Interface > Arena > Show Defensive Cooldowns

### Шаг 3. Развитие осознанности

Без аддонов, отслеживающих всё за вас, ключевыми становятся:

**Визуальное распознавание.** Учите анимации ключевых способностей. Divine Shield — золотой пузырь. Ice Block — замороженная глыба. Pain Suppression — фиолетовый щит. Это нужно знать наизусть.

**Аудиальное распознавание.** Многие заклинания имеют уникальные звуки. Polymorph, Hex, Fear — у каждого свой звук начала каста. Слушайте игру.

**Ментальные таймеры.** Тринкет противника — 120 секунд. Divine Shield — 5 минут. Icebound Fortitude — 3 минуты. Считайте в голове. Это навык, который раньше за вас выполняли аддоны.

### Шаг 4. Коммуникация
В арене с голосовой связью ваш партнёр — лучший «аддон». Вызывайте ключевые события:
- «Тринкет использован»
- «Щит на кулдауне»
- «DR на полном — станим через 3 секунды»

## Реакция сообщества

Изменения вызвали поляризацию. На форумах — тысячи постов с требованием отката. Аргументы критиков: аддоны были частью экосистемы 20 лет, их удаление снижает accessibility и повышает входной барьер.

Аргументы защитников: аддоны превратили PvP в мини-игру «смотри на панели», а не «смотри на поле боя». Ограничения возвращают навык и осознанность.

Blizzard частично отступила: некоторые ограничения были ослаблены в хотфиксах. Aренные фреймовые аддоны и DR-трекеры работают. Но автоматический трекинг вражеских кулдаунов не вернётся.

## Влияние на мету

API Lockdown косвенно меняет баланс. Спеки с «невидимыми» бёрстами (которые раньше мгновенно детектировались аддонами) стали сильнее. Классы с хорошей визуальной читаемостью (яркие анимации, очевидные касты) потеряли меньше.

Победители: Subtlety Rogue (невидимые сетапы), Feral Druid (неочевидные бёрсты), Discipline Priest (сложно отслеживать Atonement без аддонов).

Проигравшие: спеки, полагавшиеся на точное знание вражеских кулдаунов для сетапов (Arms Warrior, Frost DK).

## Итог

API Lockdown — это не конец PvP, а его перезагрузка. Порог входа вырос. Потолок мастерства — тоже. Игроки, инвестирующие в знание классов и ментальное отслеживание, получат преимущество над теми, кто полагался на аддоны. Blizzard говорит: «Смотрите на игру, а не на панели». Согласны вы или нет — адаптироваться придётся.
