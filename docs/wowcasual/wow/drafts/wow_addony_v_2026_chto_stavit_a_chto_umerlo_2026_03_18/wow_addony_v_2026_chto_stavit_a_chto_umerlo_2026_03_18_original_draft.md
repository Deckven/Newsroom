# Аддоны в 2026: что ставить, а что умерло

Вы вернулись в WoW после перерыва, запустили Midnight, а половина ваших аддонов — красная стена ошибок. Не вы один. Второго марта 2026 года Midnight вышел, а вместе с ним обрушилась значительная часть аддон-экосистемы, которая строилась пятнадцать лет. Blizzard провернули самый масштабный API-локдаун в истории WoW, одновременно встроив в базовый интерфейс функции, ради которых раньше ставили пять-шесть отдельных модов.

Этот каталог — карта нового ландшафта. Что умерло, что теперь не нужно, что по-прежнему живо и незаменимо.

## Что Blizzard встроили в игру

Прежде чем лезть в CurseForge, откройте настройки интерфейса. Blizzard в патче 12.0 натащили в дефолтный UI столько, что часть игроков впервые за десять лет играет вообще без аддонов.

**Edit Mode (расширенный).** Можно перемещать, масштабировать и перенастраивать практически любой элемент интерфейса. **MoveAnything** и **Bartender** больше не нужны для большинства игроков.

**Встроенный дамаг-метр.** Включается в Interface → Combat Tools. Показывает урон, исцеление и метрики за сегмент или данж. Для казуала — достаточно.

**Boss Timeline.** Вертикальная шкала: что босс кастовал, кастует сейчас и что будет дальше. Встроенный DBM-лайт.

**Cooldown Manager.** Нативный трекер кулдаунов: основные абилки, проки, дефы. Настраивается через Edit Mode.

**Pandemic Tracking.** Отслеживание обновления DoT-эффектов прямо в стандартных фреймах.

**Assisted Highlights.** Подсветка рекомендованных способностей — замена ротационных хелперов вроде Hekili.

## Кладбище аддонов: R.I.P.

### WeakAuras (боевые функции)

Главная потеря. Команда WeakAuras официально объявила, что не будет выпускать версию для Midnight. Причина — концепция «секретных значений» (secret values): персональные боевые данные (баффы, ресурсы, кулдауны) скрыты от аддонов. WeakAuras не может отображать то, что от него прячут.

**Замена:** встроенный Cooldown Manager, Pandemic Tracking, Boss Timeline. Для визуальных триггеров — **MidnightSimpleAuras** (упрощённый аналог в рамках новых ограничений).

### Hekili

Ротационный хелпер — невозможен в новой API-модели. Аддоны больше не имеют доступа к real-time боевым данным для принятия решений.

**Замена:** Assisted Highlights (встроенная функция).

### GTFO, TellMeWhen, MaxDps, ConRO

Вся категория боевых трекеров и ротационных помощников мертва. API-изменения в 12.0 сделали невозможным анализ боевого состояния в реальном времени.

## Must-have для всех: базовый набор

### 1. Plumber
QoL-улучшения: лучшее окно лута, подсказки по крафту, мелкие UI-фиксы. После установки думаешь: «Почему это не в игре по умолчанию?»

### 2. Better Bags (или Baganator)
Объединяет сумки в одно окно с автоматической категоризацией. Better Bags — минималистичный. Baganator — мощнее, с поддержкой банка и альтов.

### 3. Leatrix Plus
Швейцарский нож: авто-ремонт, авто-продажа мусора, ускорение диалогов, скрытие иконок миникарты. Полностью модульный.

### 4. HandyNotes + плагины для Midnight
Отмечает на карте сокровища, рарников, скрытые объекты. Для зон Midnight уже есть плагины. Без него пробежите мимо половины контента.

### 5. Auctionator
Заменяет дефолтный интерфейс аукционного дома на быстрый и удобный.

### 6. Enhance QoL
Огромное количество мелких улучшений под одной крышей. Дополняет Leatrix Plus.

## Must-have для рейдеров и M+

### 1. BigWigs + LittleWigs
Главный боссмод после API-локдауна. Перешёл на официальные данные Blizzard (Boss Timeline API). Таймеры, звуковые предупреждения, алерты. Альтернатива — **DBM** (тоже адаптирован).

### 2. Details! Damage Meter
Пережил аддон-апокалипсис. Побарный разбор, разбивка по заклинаниям, сравнение между попытками. Данные сохраняются между сессиями — в отличие от встроенного метра.

### 3. Platynator (или Plater)
Кастомные неймплейты — необходимость в M+. Настраиваемые дебаффы, баффы, угроза и кастбары на неймплейтах врагов.

### 4. Method Raid Tools (MRT)
Трекинг рейдовых кулдаунов, экстерналов, заметки для боссов, назначения. Работает, потому что оперирует на уровне координации, а не боевой автоматизации.

### 5. Angry Keystones
Расширенные таймеры для М+: время на каждого босса, процент мобов, время до деплита. Делает одну вещь и делает её хорошо.

### 6. Pawn
Быстрая оценка предметов: апгрейд ли это для вашего билда. Поддерживает кастомные стат-веса.

## Nice-to-have: приятные мелочи

**Altoholic** — панель управления зоопарком альтов: золото, сумки, профессии, кулдауны по всем персонажам.

**CraftSim** — симулятор крафта: вероятности качества, оптимальные реагенты, потенциальная прибыль.

**HomeBound** — чеклист декораций для хаузинга с 3D-превью каждого предмета.

**Dialogue UI** — перерабатывает окно диалогов с NPC. Текст квестов становится читабельным.

**Rarescanner** — сканирует зону на рарников и сокровища, уведомляет при спавне.

**Midnight Routine** — трекер еженедельного прогресса: Great Vault, ключи, локауты, еженедельные квесты.

**AllTheThings** — для коллекционеров: показывает абсолютно всё, что можно собрать в игре.

## PvP в Midnight: API-локдаун без исключений

Blizzard чётко заявили: ограничения на боевые аддоны в PvP работают без исключений.

**Что это значит:**
- **OmniCD и OmniBar** — жёстко ограничены. Трекинг вражеских кулдаунов интеррапт-способностей не работает в реальном времени
- **GladiusEx / sArena** — базовые арена-фреймы остались, глубокий трекинг кулдаунов урезан
- «Фейк-кастинг» стал навыком интуиции, а не чтения таймера
- Помнить кулдауны противника — теперь вопрос памяти, а не автоматического трекера

Для PvE Blizzard ослабили часть ограничений: самые жёсткие рестрикции — только во время босс-энкаунтеров и в М+. Для рейтингового PvP послаблений не было.

## Аддон-менеджеры

### CurseForge (Standalone)
Крупнейший каталог. Standalone-клиент без лишнего bloatware: установка, обновление, бэкапы. **Основной менеджер для большинства.**

### WowUp
Лёгкий, быстрый, без рекламы. Поддерживает WoWInterface и GitHub. Нет прямой интеграции с CurseForge. **Хороший запасной вариант.**

### Ajour
Опенсорсный на Rust. Самый быстрый, полностью бесплатный. **Для тех, кому важен open-source.**

**Рекомендация:** CurseForge Standalone как основной + WowUp для аддонов с WoWInterface/GitHub.

## Советы для возвращающихся

1. **Начните с чистого листа.** Не тащите старый набор. Включите дефолтный UI, поиграйте час, поймите, чего не хватает, и только потом ставьте аддоны точечно.

2. **Минимум аддонов на первый заход.** Platynator и BigWigs. Добавляйте остальное по одному.

3. **BugSack + BugGrabber** — ставьте всегда. Перехватывают Lua-ошибки вместо спама на экране.

4. **Не ищите замену WeakAuras.** Её нет. Философия изменилась: Blizzard хотят, чтобы вы пользовались встроенными инструментами.

5. **Проверяйте дату обновления.** Если последний апдейт старше января 2026 — аддон, скорее всего, не работает. API-изменения в 12.0 были слишком масштабными.

6. **Account Wide Interface Settings** — синхронизирует настройки UI между персонажами. Экономит часы на альтах.

7. **Не паникуйте из-за PvP-ограничений.** Все в одинаковых условиях. Через пару недель адаптируетесь.

*Мир аддонов в 2026 году — не катастрофа, а перезагрузка. Blizzard забрали у аддонов власть над боевой информацией и взяли ответственность на себя. Дефолтный UI Midnight — первый за двадцать лет, с которым реально можно жить. А то, что аддоны по-прежнему делают лучше — сумки, карты, координация рейда — никуда не делось. Просто список стал короче. И это, возможно, хорошо.*
