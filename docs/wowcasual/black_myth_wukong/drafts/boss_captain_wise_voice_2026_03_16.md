# Как победить Captain Wise-Voice — Black Myth: Wukong

Огромная химера из сросшихся зверей, стреляющая лазерами из головы и уполовинивающая вашу максимальную полоску здоровья. Captain Wise-Voice — самый сильный из четырёх капитанов Пагоды и обязательный босс для продвижения в третьей главе. Бой медленный, но требует понимания одной неочевидной механики: у этого босса повреждаются только определённые части тела.

## Кто такой Captain Wise-Voice

Captain Wise-Voice (魔将·妙音, Мо цзян Мяоинь — «Демон-генерал: Божественная Мелодия») — Yaoguai King, гигантское существо-амальгамация из нескольких зверей. Сильнейший из четырёх капитанов Пагоды Realm.

Дух Captain Wise-Voice — один из четырёх духов капитанов, необходимых для квестовой линии **Третьего Принца** (Third Prince), который находится в Lower Pagoda, в камере рядом с начальной точкой.

## Локация

**Pagoda Realm, Mani Wheel** — третья глава. От святилища перейдите по мосту в форме руки внутрь здания. Босс ждёт наверху.

Тип босса: **Yaoguai King**. Обязательный для продвижения сюжета.

## Ключевая механика: уязвимые точки

**Тело Captain Wise-Voice неуязвимо.** Обычные удары по корпусу не наносят урона. Повреждаются только:

| Точка | Эффект |
|-------|--------|
| **Голова (жёлтое ядро)** | Основная цель. Максимальный урон. Трудно достать, пока босс стоит. |
| **Треснувшая рука (Cracked Arm)** | Удары по ней наносят средний урон и постепенно **вызывают падение босса**, открывая голову для атаки. |
| **Ноги** | Удары по ногам стаггерят босса. При стаггере он падает, и голова становится доступной. |

**Переключайте захват камеры** между головой и треснувшей рукой по ситуации. Когда босс падает — немедленно бейте по жёлтому ядру на голове.

## Атаки и паттерны

Captain Wise-Voice — очень медленный босс с огромным запасом HP и умеренным уроном. Опасность не в скорости, а в AoE-атаках и дебаффе здоровья.

**Laser Rampage (лазерный шквал).** Стоя неподвижно, выпускает множество случайных лазерных лучей из головы. **Непрерывно перекатывайтесь** для минимизации попаданий. Атака начинается в самом начале боя, когда вы приближаетесь.

**Heavy Laser Beam (тяжёлый лазер).** Заряжает золотую энергию в голове и выпускает мощный одиночный луч. Замах длинный — готовьте уклонение во время зарядки.

**Stomp (топот).** Подпрыгивает и топает ногами, создавая малое AoE вокруг точки приземления. Отступайте сразу после прыжка.

**Charged Electricity Stomp (электрический топот).** Заряжает электричество в руке, затем бьёт по земле, создавая **круговой электрический ток на земле**. Подпрыгните или отбегите до удара по земле.

**Electricity Rain (электрический дождь).** Вызывает множество электрических разрядов сверху. На земле появляются **круговые индикаторы** в местах попадания. Уходите из отмеченных зон.

**Сокращение максимального HP.** Особая механика — некоторые атаки временно **урезают вашу максимальную полоску здоровья вдвое**. Эффект проходит через время, но пока он действует — вы крайне уязвимы.

## Рекомендуемый билд

### Заклинания

**Immobilize** (обязательно) — босс медленный, но его огромный HP требует длинных окон для урона. Замораживайте и бейте по треснувшей руке или голове.

**A Pluck of Many** (рекомендуется) — клоны наносят постоянный урон, пока вы уклоняетесь от лазеров. Особенно полезно, так как босс много стоит на месте.

### Дух

**Wandering Wight** — стаггер помогает опрокинуть босса, открывая голову.

### Тактика

1. Атакуйте ноги для стаггера → босс падает
2. Переключите захват на голову (жёлтое ядро) → максимальный урон
3. Когда босс встаёт — бейте по треснувшей руке для следующего опрокидывания
4. Повторяйте цикл

## Типичные ошибки

1. **Бьёте по корпусу.** Тело неуязвимо. Только голова, треснувшая рука и ноги.

2. **Не переключаете захват камеры.** Lock-on по умолчанию целится в корпус. Вручную переключайте на голову или руку.

3. **Стоите в зонах Electricity Rain.** Индикаторы на земле — ваши подсказки. Выходите из кругов.

4. **Паникуете при сокращении HP.** Эффект временный. Не тратьте расходники панически — полоска восстановится.

5. **Жадничаете при Laser Rampage.** Лучи случайные и покрывают большую площадь. Непрерывные перекаты — единственная защита.

## Зачем этот бой важен

Дух Captain Wise-Voice — один из четырёх, необходимых для Третьего Принца. Без него вы не завершите квестовую линию Pagoda Realm. Сам бой учит важному навыку: прицельной атаке по уязвимым точкам вместо бездумного нанесения урона. Этот навык пригодится в нескольких поздних боях, где у боссов тоже есть бронированные и уязвимые зоны.
