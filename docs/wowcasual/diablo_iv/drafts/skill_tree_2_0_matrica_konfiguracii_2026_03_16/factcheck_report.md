## Проблемные факты

### ⚠ Матрица 3×2×2 — авторская интерпретация

Центральная концепция статьи — формула «3×2×2 = 12 конфигураций на навык» — является авторской систематизацией, а не официальным описанием Blizzard. Реальная система может отличаться по структуре, хотя общая идея большей вариативности подтверждена.

### ⚠ Спекулятивные примеры навыков

Конкретные примеры модификаций навыков (числовые значения, названия модификаторов) основаны на предположениях и могут не соответствовать финальной реализации.

## Рекомендуемые исправления

- Пометить формулу «3×2×2» как авторскую интерпретацию / модель, а не официальное описание механики.
