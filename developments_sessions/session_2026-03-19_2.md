#### Технические уроки Todoist API:
- `gatekeeper todoist push` не умеет создавать вложенные подзадачи (>1 уровень) — использовать API v1 напрямую
- REST v2 возвращает 410 для некоторых проектов — использовать только v1
- Перемещение задач между проектами: `POST /api/v1/tasks/{id}/move` (не update)
- Лимит задач на проект: ~300 (Free/Pro)
- Rate limit: `time.sleep(0.12-0.15)` между запросами
