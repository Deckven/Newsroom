### 4. Фреймворк Protogons — анализ и интеграция
- Расположение: `M:\AIZoo\Protogons\Protogons\`
- Описание: CLI-фреймворк для оркестрации пайплайна AI-агентов
- Пайплайн из 4 агентов:
  1. **Preprocessor** — анализ темы, извлечение ключевых концепций, аудитории, ограничений
  2. **Instructor** — генерация полного system prompt для нового AI-ассистента
  3. **Umberto** — семиотический ревью промпта (оценка качества, score/10)
  4. **OTK** — тестирование промпта, вердикт (pass / needs_revision)
- Feedback loop: если OTK дает needs_revision, пайплайн перезапускает Instructor->Umberto->OTK
- Конфиг: `protogons.toml` (TOML), поддержка OpenAI/Anthropic/Gemini провайдеров
- Выход (InstructorOutput): JSON с полями agent_name, system_prompt, role_description, capabilities, limitations, example_interactions
