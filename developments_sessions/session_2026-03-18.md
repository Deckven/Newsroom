## Решение по методологии исследования

Агенты использовали **только веб-поиск**, минуя курированные YAML-источники из `sources/wowcasual/wow_sources/` (80+ файлов). Это было ошибкой — источники проекта простаивали.

**Новый порядок для будущих сессий:**
1. Прочитать YAML-файлы из `sources/{domain}/{topic}_sources/` — извлечь URL
2. WebFetch по релевантным курированным источникам
3. Только после этого — WebSearch для дополнения
4. В frontmatter отмечать происхождение источников

Сохранено в `feedback_research_sources_first.md`.
